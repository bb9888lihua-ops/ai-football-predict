## 🚀 部署步骤

### 1. 创建Firebase项目（5分钟）
- 访问 Firebase 控制台
- 创建项目 `ai-football-predict`
- 启用 Firestore 数据库

### 2. 部署网站（3分钟）
```bash
cd football-predict
./deploy.sh
```

### 3. 导入示例数据（2分钟）
- 登录后台 `admin` / `admin888`
- 导入 `sample_matches.json`

### 4. 设置自动预测（10分钟）
- 下载服务账号密钥
- 安装Python依赖
- 运行 `./setup_cron.sh`

**总耗时**: 约20分钟
